# EduGame Platform

Plataforma Educacional Gamificada desenvolvida com FastAPI, Streamlit e PostgreSQL/Supabase.

## Inicio Rapido

1. Clone o repositorio
2. Instale as dependencias: `pip install -r requirements.txt`
3. Configure o arquivo `.env` com suas credenciais Supabase
4. Execute o backend: `uvicorn main:app --reload`
5. Execute o frontend: `streamlit run Home.py`

## Documentacao Completa

Veja [docs/API.md](docs/API.md) para documentacao detalhada da API e arquitetura.

## Funcionalidades

- Sistema de Desafios com submissoes e votacao
- Mini-Provas com cronometro e correcao automatica
- Gamificacao com pontos, niveis e medalhas
- Formacao de Equipes automatica
- Missoes Educacionais
- Enquetes em tempo real
- Interface Dark Theme moderna
